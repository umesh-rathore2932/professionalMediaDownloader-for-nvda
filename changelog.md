## Version 2.0
* **Auto-Clipboard Fetch:** The toolkit now automatically detects a URL from the clipboard upon opening and announces it via speech.
* **Expanded Localization:** Full localization support added for Russian (ru) and Ukrainian (uk).
* **Audio Feedback System:** Integrated winsound beeps to provide real-time status updates (Start, Success, and Error cues).
* **Manual Paste Option:** Added a dedicated button to manually paste URLs from the clipboard for better user control.
* **Context-Aware Help:** The Help menu now automatically opens the documentation (readme.html) in the user's active NVDA language.
* **Backend Optimization:** Switched to FFprobe for more efficient metadata extraction and improved background process performance.
