# Професійний завантажувач медіа #

* Автор: Умеш Раторе
* Версія: 1.0

Професійний завантажувач медіа — це високопродуктивний та доступний інструмент, розроблений для користувачів NVDA, що дозволяє завантажувати аудіо- та відеоконтент із понад 1 000 стрімінгових платформ, зокрема YouTube, Facebook, Instagram та інших.

## Основні комбінації клавіш ##
* **NVDA+Shift+Y**: відкриває діалог додатка з будь-якої програми.
* **Alt+D**: переміщає фокус на кнопку **Завантажити**.
* **Alt+O**: відкриває папку **Завантаження** у Провіднику Windows.

## Як користуватися ##
1. Скопіюйте URL медіафайлу, який ви хочете завантажити, з браузера.
2. Натисніть **NVDA+Shift+Y**, щоб запустити інтерфейс завантажувача.
3. Вставте скопійовану URL-адресу в поле **Ввести URL**.
4. За допомогою клавіші **Tab** перейдіть до поля **Вибір формату** і виберіть бажаний формат (наприклад, MP3, MP4, OGG).
5. Натисніть **Enter** або клацніть кнопку **Завантажити**, щоб розпочати процес.
6. З'явиться доступний діалог перебігу виконання. Ви отримаєте сповіщення, коли завантаження буде завершено.

## Функції та підтримувані формати ##
* **Аудіоформати**: MP3, WAV, M4A, OGG, FLAC.
* **Відеоформати**: MP4, MKV, WEBM (висока роздільна здатність).
* **Інтеграція**: Безперешкодно інтегрується в меню «Інструменти» NVDA.
* **Організація**: Автоматично сортує файли у спеціальні папки «Аудіо» та «Відео».

## Технічні вимоги ##
Для обробки та конвертації цьому додатку необхідні файли `yt-dlp.exe`, `ffmpeg.exe` та `ffprobe.exe`, які мають бути присутніми у каталозі додатка.
